## üéØ How It Works

### Generation Pipeline

```mermaid
User clicks topic
    ‚Üì
[CLEANUP] Remove old p5 instances, stop animations, kill GSAP tweens
    ‚Üì
[PART 1] Generate HTML (/generate-html)
    ‚Üí AI creates structure with #visual-demo div
    ‚Üí Inject HTML, wait 300ms
    ‚Üì
[PART 2] Generate CSS (/generate-css)
    ‚Üí AI creates themed styles with media queries
    ‚Üí Inject CSS, wait 300ms
    ‚Üì
[PART 3] Generate Animation (/generate-animation)
    ‚Üí AI creates responsive JavaScript with library code
    ‚Üí Execute in isolated scope
    ‚Üì
[PART 4] Stream Content (/generate-content)
    ‚Üí AI streams brief article text
    ‚Üí Real-time markdown rendering
    ‚Üì
Display related topics ‚Üí Repeat cycle
```

### Key Endpoints

1. **`/generate-initial-ui`** - Creates 6 random starting topics
2. **`/generate-html`** - Generates responsive HTML structure
3. **`/generate-css`** - Generates themed CSS with mobile queries
4. **`/generate-animation`** - Generates interactive JavaScript
5. **`/generate-content`** - Streams article text via SSE
